# EduShield AI 🎓

## Student Dropout Risk Prediction System

EduShield AI is a Machine Learning-based web application that predicts whether a student is at High Risk or Low Risk of dropping out based on academic and personal factors.

## Features
- Predicts student dropout risk
- User-friendly Streamlit interface
- Random Forest Machine Learning model
- Instant prediction results
- Recommendations based on prediction

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib

## Input Features
- Age
- Education
- Attendance
- Percentage
- Study Hours
- Failed Subject
- Course Interest
- Financial Difficulty
- Distance
- Internet Access
- Social Media Hours
- Missed Classes
- Break Thought

## Model
Random Forest Classifier
